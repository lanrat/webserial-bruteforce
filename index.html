Information

This page uses WebSerial to try every possible baud rate and returns the one that returns the most ASCII data.

This is useful when reversing a new device and the UART configuration is unknown.

For now only the most common baud rates are tested and no other modes are set. A future version my include tests for other settings and allow for sending simulated data as well.
